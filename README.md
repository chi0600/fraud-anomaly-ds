# fraud-anomaly-ds
# Fraud / Anomaly Detection (Data Science Project)

A compact, end-to-end data science project focused on fraud detection and anomaly analysis.

## Goals
- Understand highly imbalanced data and define the right evaluation metrics (PR-AUC, precision/recall)
- Build strong baselines and compare modeling approaches
- Provide interpretable insights and a reproducible workflow

## Project Structure
- `notebooks/01_eda.ipynb` – Exploratory data analysis and initial observations
- `requirements.txt` – Python dependencies
- `data/` – Local data (ignored by git)

## How to Run
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
